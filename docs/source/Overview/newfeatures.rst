New Features
============

Omnia v1.6.1 addresses a particular issue. It involves the unavailability of the dependent package 'libssl1.1_1.1.1f-1ubuntu2.22_amd64' required by Omnia v1.6 for the Ubuntu 22.04 operating system. The focus of this release is to resolve this issue and ensure the proper functionality of Omnia on Ubuntu 22.04 OS.
